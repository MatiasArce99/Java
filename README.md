# Variables primitivas
___
## Enteros
- ***Int:*** 4 bytes de espacio de almacenamiento. Desde -2 mill hasta 2 mill.
- ***Short:*** 2 bytes de espacio de almacenamiento. Desde -32 mil hasta 32 mil.
- ***Long:*** 8 bytes de espacio de almacenamiento. Gran almacenamiento.
- ***Byte:*** 1 byte de espacio de almacenamiento. Desde -128 hasta 127.
___
## Flotantes (decimales)
- ***Float:*** 4 bytes de almacenamiento.
- ***Double:*** 8 bytes de almacenamiento.
___
## Strings
- ***Char:*** Representa caracteres.
___
## Booleanos
- ***Boolean:*** Respresenta solo dos valores "true" y "false".
___
## Definición de variable
- Espacio en memoria donde se almacenará un valor que puede cambiar durante la ejecución.
- En JAVA es importante inicializar todas las variables.
